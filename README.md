# Recruitment_Text_Mining
채용공고 텍스트 마이닝과 생성형AI를 통한 취업 인사이트 도출 및 제공 

## ✍ 요약
- SBERT와 비지도학습을 기반으로 채용공고의 문장단위 분석을 진행
- KeyBERT, BERTopic을 활용하여 군집 해석 진행
- Django Framework를 활용한 WebPage 구축 및 생성형 AI를 활용하여 통해 분석 결과의 활용성을 높혔다.
